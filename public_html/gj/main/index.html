What is Game Jam?

Game Jam is a contest where participants utiliz their skills and knowledges to make a game from scratch with a time constraint.

Participants might work independently, or in teams.

Participants are generally programmers, game designers, artists, writers, and others in game development-related fields.

Traditionally, game jams focus on video games; however, board games have also been the subject of game jams.
